# Retail Sales Analysis Dashboard

## Project Title
Retail Sales Analysis using Power BI

## Short Description / Purpose
This project analyzes retail sales data to understand how the business is performing.
The goal of the project is to find useful insights such as which region has the highest sales, which product categories generate the most profit, and how sales change over time.
The data was first cleaned and prepared using **Python in Jupyter Notebook**, and then an **interactive dashboard was created in Power BI** to visualize the results. This helps users easily understand the sales performance and make better business decisions.

## Tech Stack
The following tools were used in this project:
* **Python (Pandas)** – for cleaning and preparing the dataset
* **Jupyter Notebook** – for data preprocessing and exploration
* **Power BI** – for creating interactive dashboards and visualizations

## Data Source
The dataset used in this project is the **Superstore Sales Dataset**.
It contains information about retail sales transactions, including:
* Order Date
* Sales
* Profit
* Quantity
* Discount
* Region
* Category
* Sub-Category
* Customer Segment
This dataset is commonly used for learning data analysis and visualization.

## Features
This project includes the following analysis and dashboard features:
* Total Sales and Total Profit summary
* Monthly sales trend analysis
* Sales comparison across different regions
* Profit analysis by product category
* Sales distribution across sub-categories
* Discount vs Profit analysis
* Interactive filters to explore data by region, category, and year
The dashboard allows users to interact with the data and easily explore business performance.

## Project Outcome
Through this project, sales data was analyzed and visualized to understand business trends and performance. The dashboard helps identify high-performing regions, profitable product categories, and patterns in sales over time.
This project demonstrates how data cleaning, analysis, and visualization can be used to turn raw data into meaningful business insights.
